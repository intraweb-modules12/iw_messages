//************************
// Files structure
//************************
This module needs the iw_main module installed at least in the version 1.2, so you must install or upgrade it if you don't have it. The iw_main module contains the general libraries and functions used in Intraweb modules.

The attached files must be updated to the server, so a folder with written permissions is required. This folder can be out of the public html pages and it is possible to define exactly where it is situated from the module options.

//************************
// Module installation
//************************
Locate the files of the modules iw_main and iw_messages into the modules directory, install and active them following the usual methods used in PostNuke modules:

	- Regenerate modules list
	- Initialize modules or upgrade iw_main
	- Active modules

//************************
// Module configuration
//************************
After the module installation it is necessary to define what groups are allowed to send messages to full groups and who can attach files in the messages.
